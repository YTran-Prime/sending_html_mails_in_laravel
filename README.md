## Prerequisites
<ul>
<li>After cloning this repository, go to the root folder, run the following command/s,
<pre>
    composer install
    composer update</pre>
</li>
<li>Rename .env.example to .env and provide your sendgrid details there.</li>
<li>Run <pre>php artisan key:generate</pre> </li>

</ul>
